## Roadmap

Our goal is to implement Scala equivalents of popular Python LLM frameworks:

- [ ] Single API access to multiple LLM providers (like [LiteLLM](https://www.litellm.ai/))
- [ ] A comprehensive toolchain for building LLM apps (like [LangChain](https://www.langchain.com/)/[LangGraph](https://www.langchain.com/langgraph))
- [ ] An agentic framework (like [PydanticAI](https://ai.pydantic.dev/), [CrewAI](https://www.crewai.com/))
- [ ] Tokenization utilities (port of [tiktoken](https://github.com/openai/tiktoken))
  - https://cookbook.openai.com/examples/how_to_count_tokens_with_tiktoken
  - https://github.com/knuddelsgmbh/jtokkit
  - https://github.com/hyunwoongko/gpt2-tokenizer-java
- [ ] Full ReAct loop implementation
- [ ] Simple tool calling mechanism

See:
- https://github.com/langchain4j/langchain4j
- https://spring.io/projects/spring-ai
